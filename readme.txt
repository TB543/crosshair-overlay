How To Use:

1. press f2 to toggle program on/off (when run it will default to off)
2. press f3 to make the crosshair bigger
3. press f4 to make crosshair smaller
4. to change the symbol of the crosshair open the program with a text editor and on line 22 change the symbol in the parenthesis
5. in order for crosshair to show up in games make sure the game is set to borderless window, it will not work when fullscreen
